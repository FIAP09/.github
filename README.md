# FIAP09 - Engenharia de Software

## Sobre a Organização

Esta organização é dedicada aos projetos realizados pela turma de Engenharia de Software da FIAP.

## Objetivo

O objetivo desta organização é centralizar e compartilhar os projetos desenvolvidos durante o curso, promovendo o aprendizado colaborativo e a aplicação prática dos conceitos de engenharia de software.

## Projetos

Os repositórios desta organização contêm trabalhos acadêmicos, projetos práticos e exercícios desenvolvidos pelos alunos ao longo do curso de Engenharia de Software.
